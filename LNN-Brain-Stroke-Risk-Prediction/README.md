# 🧠 Brain Stroke Risk Prediction Using Machine Learning and Explainable AI

A healthcare analytics project that predicts the likelihood of stroke occurrence using demographic, lifestyle, and clinical factors. The project combines advanced machine learning techniques with explainable AI approaches to improve prediction accuracy and model interpretability.

---

## 1. Project Overview

Stroke is one of the leading causes of death and long-term disability worldwide. Early identification of high-risk individuals can support preventive healthcare interventions and improve patient outcomes.

This project develops a binary classification system to predict stroke risk using patient health records and compares multiple machine learning and deep learning models.

---

## 2. Key Features

### Data Preprocessing

* Missing value handling using median imputation
* Removal of non-predictive identifiers
* Feature scaling and transformation
* Data quality validation

### Feature Engineering

* Binary encoding of categorical variables
* One-hot encoding for nominal features
* Clinical attribute transformation for model compatibility

### Class Imbalance Handling

* SMOTE (Synthetic Minority Over-sampling Technique)
* Improved minority class representation
* Enhanced model generalization

### Explainable AI

* SHAP-based feature importance analysis
* Model interpretability for healthcare decision support
* Transparent prediction insights

## 3. Dataset Features

The dataset contains patient demographic and medical information including:

### Demographic Information

* Gender
* Age
* Marital Status
* Residence Type
* Work Type

### Clinical Factors

* Hypertension
* Heart Disease
* Average Glucose Level
* Body Mass Index (BMI)

### Lifestyle Factors

* Smoking Status

### Target Variable

* Stroke Occurrence (0 = No Stroke, 1 = Stroke)
